# 🚗 Rayka Rental

A simple and modern vehicle rental system designed for **FiveM**, supporting **QBCore**, **ESX**, **QBOX**, and **Standalone** servers.

---

## ✨ Features

* Clean and modern UI
* Fully configurable through `config.lua`
* Supports **Target System**
* Built-in **Interact System**
